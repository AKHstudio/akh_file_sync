🚀 クイックスタートガイド

akhsyncを使ったアドオン開発をすぐに始める

1. npm の初期化

まず、新しいプロジェクトを初期化します：

npm init your_project_name -y

このコマンドで package.json が自動的に作成されます。

2. ファイル構造の作成

アドオン開発プロジェクトのルートディレクトリ構造は以下のようになります：

your_project_root |-- src/ (開発用ソースコード) |-- addon1/ |-- behavior_packs/ | |-- scripts/ | |-- main.js |-- resource_packs/ |-- addon2/ (複数のアドオンを開発する場合 - オプション) |-- behavior_packs/ | |-- scripts/ | |-- main.ts |-- resource_packs/ |-- world/ (ビルド用のワールドデータ - オプション) |-- tsconfig.json (TypeScriptを使用する場合のみ) |-- package.json

この構造に従ってディレクトリとファイルを作成してください。

3. 必要なライブラリのインストール

使用したいバージョンに応じて @minecraft ライブラリをインストールします。

バージョン一覧：

  • @minecraft/server
  • @minecraft/server-ui

以下は最新の安定版をインストールする例です。

akhsync

npm install @akhstudio/akhsync@latest --save-dev

@minecraft/server

npm install @minecraft/server@latest --save-dev

@minecraft/server-ui (UI を使用する場合)

npm install @minecraft/server-ui@latest --save-dev

3.1 TypeScript を使用する場合は tsconfig.json を作成 (オプション)

⚠️ IMPORTANT

TypeScript を使用しない場合、このステップは不要です。

📌 NOTE

@minecraft/math モジュールを使用する場合は、paths オプションを適宜調整してください。

以下の設定は例です。プロジェクトに合わせて調整してください。

{ "compilerOptions": { "target": "ES2020", "module": "esnext", "moduleResolution": "bundler", "baseUrl": "./", "paths": { "@minecraft/math": ["node_modules/@minecraft/math/dist/minecraft-math.d.ts"] }, "resolvePackageJsonImports": true, "allowJs": true, "noEmit": true, "esModuleInterop": true, "forceConsistentCasingInFileNames": true, "strict": true, "skipLibCheck": true }, "include": ["**/scripts/**/*"], "exclude": ["node_modules", "build", "dist"] }

4. package.json にスクリプトを追加

package.json の scripts セクションに以下を追加します：

{ "scripts": { "build": "akhsync build", "sync": "akhsync sync", "async": "akhsync async", "watch": "akhsync watch", "dist": "akhsync dist" } }
💡 TIP

追加のオプションを指定したい場合は、-- の後に追加してください。

npm run build -- --development

次のステップ

セットアップが完了しました！次は実際にコマンドを使ってアドオン開発を始めましょう。

📌 NOTE

ローカルインストールの場合、npm run を使わずに npx akhsync コマンドで直接実行することもできます。

例: npx akhsync build または npx akhsync watch
